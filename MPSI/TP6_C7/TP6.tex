\end{itemize}			
\item \textbf{Modéliser} :
\begin{itemize}
\item Identifier et caractériser les grandeurs physiques : caractéristiques fréquentielles

\item Signaux canoniques d'entrée : signaux sinusoïdaux
\item Schémas blocs, fonctions de transferts
\end{itemize}

\begin{itemize}
\item Réponse fréquentielle : systèmes du 1\up{er} et 2\up{ème} ordre, intégrateur.
\item Paramètre de résolution numérique
\item Grandeurs simulées
\end{itemize}	
\item \textbf{Communiquer : } Rechercher et traiter des informations techniques

\begin{itemize}
\item S'approprier le fonctionnement d'un système pluritechnologique : paramètres influents
\item Proposer et justifier un protocole expérimental : modèles de comportements d'un systèmes
\item Mettre en oeuvre un protocole expérimental : Identification temporelle et fréquentielle d'un modèle de comportement.
